# 🤖 How To Implement ChatGPT in Django

## 📌 Project Description

This project is a simple AI Chatbot developed using Python and Django. It allows users to ask predefined questions and receive appropriate responses through a clean chat interface.

## 🚀 Features

- Professional Chat Interface
- User & Bot Chat Bubbles
- Chat History
- Clear Chat Button
- Django Backend
- Python Logic
- Responsive Design

## 🛠️ Technologies Used

- Python 3.10
- Django 5.2.15
- HTML
- CSS
- JavaScript
- VS Code

## 📂 Project Structure

```
chatgpt_project/
│
├── chatbot/
├── chatgpt_project/
├── templates/
├── manage.py
├── db.sqlite3
├── requirements.txt
└── README.md
```

## ▶️ How to Run

1. Install Python
2. Install Django

```
pip install -r requirements.txt
```

3. Run the server

```
python manage.py runserver
```

4. Open

```
http://127.0.0.1:8000
```
